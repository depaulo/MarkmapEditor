Welcome to Markmap Editor


            

              A Markdown-first workspace for writing, mapping, reviewing, and exporting your ideas.
            

          


          
        


        

          

            
Markdown + Mindmap + Preview

            

              Write in Markdown, view your structure as an interactive mindmap, and switch to
              preview or export views when you need a cleaner reading or publishing mode.
            

          


          
Choose the mode that fits your work:


          

            

              
✏️


              

                
Editor

                

                  Use Editor mode for standalone Markdown files, mindmaps, outlines, notes, and
                  general writing.
                

              

            


            

              
📓


              

                
Journal

                

                  Use Journal mode for workspace-based writing with journals, concepts, search,
                  backlinks, tags, open tasks, and archive.
                

              

            


            

              
📊


              

                
Pandoc / Slides

                

                  Use Pandoc / Slides mode for presentation-oriented Markdown and export workflows.
                

              

            

          


          

            You can change modes anytime using the Mode selector in the toolbar.
